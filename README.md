# Dialog Cookies Project

This project demonstrates how to use JavaScript to manage dialogs and cookies on a web page. Users can interact with two dialogs: one for accepting information and another
for setting preferences. The preferences are saved as cookies and can be customized by the user.

## Live Demo
You can check out the live demo of the **Dialog Cookies** website here: 
[Dialog Cookies Demo](https://rahaf-ayn-kraie.github.io/dialog-cookies/)

## Features

- **Dialog Interaction**: 
  - `dialog-1`: Displays general information.
  - `dialog-2`: Allows users to set preferences.
- **Cookie Management**: 
  - Saves browser, operating system, screen width, and height details.
  - Stores user preferences based on checkbox selections.
- **Dynamic UI**: 
  - Shows/hides dialogs using JavaScript.
  - Applies overlay effects for better user experience.
